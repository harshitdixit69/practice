# Tello

Golang interface for controlling the DJI Tello drone using your keyboard.

## Getting started

Installing the dependencies:

```bash
go mod download
```

Running the application:

```bash
go run main.go
```

## Keys

```bash
 A - Going left
 D - Going right
 W - Going up
 S - Going down
 U - Going forward
 J - Going backward
 K - Rotating counter-clockwise
 H - Rotating clockwise
 T - Take off
 L - Land drone 
```
